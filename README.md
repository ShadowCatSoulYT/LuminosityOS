# LuminosityOS üí°

LuminosityOS is a GUI-based operating system inspired by KolibriOS.  
This is a **passion project**, not a finished product, and not something you should daily-drive (yet ;]).

The main idea is to create a lightweight, everyday usable OS that gives users real freedom (for example you can modify system files freely, just.. dont mess up ok?),
without unnecessary complications or artificial limitations.

## Current State
Right now, LuminosityOS is in early early early early testing.
